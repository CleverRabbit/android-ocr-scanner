# Contract Scanner - Приложение для сканирования номера договора

## Описание
Приложение для Android, которое сканирует номер договора с камеры в реальном времени и открывает ссылку с этим номером.

## Технологии
- **Язык**: Kotlin
- **Камера**: CameraX (поток в реальном времени)
- **OCR**: Google ML Kit Text Recognition (работает офлайн, быстро)

## Функционал

### Логика работы
1. При открытии включается камера (Preview)
2. В реальном времени сканируется текст с кадра
3. Фильтрация по Regex: `№\s*\d+/?\d*` (ищет номера типа № 123/456, №123, и т.д.)
4. Визуализация: При обнаружении номера показывается статус
5. Стабилизация: Номер должен быть обнаружен несколько раз подряд для подтверждения

### Режимы работы
- **Вариант А (Авто)**: Как только номер найден и подтвержден, сразу открывается браузер по шаблону URL
- **Вариант Б (По кнопке)**: Показывается найденный номер крупно внизу экрана + кнопка «Открыть»

### Настройки
В настройках приложения можно изменить:
- **Шаблон ссылки**: Например, `https://mysite.ru/contract/{NUMBER}` или `https://crm.company.ru/doc/{VAL}`
- **Авто-открытие**: Включить/выключить автоматическое открытие браузера

## Сборка и запуск

### Требования
- Android Studio Arctic Fox или новее
- JDK 17
- Android SDK 34
- Минимальная версия Android: API 24 (Android 7.0)

### Шаги
1. Откройте проект в Android Studio
2. Дождитесь синхронизации Gradle
3. Подключите устройство или запустите эмулятор
4. Нажмите Run (Shift+F10)

### Разрешения
Приложение запрашивает разрешение на использование камеры при первом запуске.

## Важные особенности

1. **Фото не сохраняются**: Приложение работает только с потоком памяти, изображения не сохраняются в галерею
2. **Офлайн работа**: ML Kit Text Recognition работает без интернета после первой загрузки модели
3. **Стабилизация обнаружения**: Номер должен быть распознан 3 раза подряд для подтверждения
4. **URL кодирование**: Номер договора автоматически кодируется для безопасной передачи в URL

## Регулярное выражение

По умолчанию используется паттерн: `№\s*\d+/?\d*`

Он находит:
- `№ 123/456`
- `№123/456`
- `№ 123`
- `№123`

Для изменения паттерна отредактируйте переменную `contractPattern` в `MainActivity.kt`.
